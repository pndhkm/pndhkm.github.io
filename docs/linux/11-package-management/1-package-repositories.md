# Package Repositories

Software repositories are indexed sources containing signed packages and metadata used by package managers.

### Check configured repositories

Input:

```
cat /etc/apt/sources.list
```

This shows active APT repositories on Debian/Ubuntu systems.

References:

* [https://man7.org/linux/man-pages/man5/sources.list.5.html](https://man7.org/linux/man-pages/man5/sources.list.5.html)
