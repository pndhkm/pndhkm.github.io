# Snap

Snap packages are containerized applications managed by the `snapd` service.

### List installed snaps

Input:

```
snap list
```

Output:

```
Name     Version  Rev  Tracking       Publisher   Notes
core20   202401   2260 latest/stable  canonical✓  base
snapd    2.63     20671 latest/stable  canonical✓  snapd
```

This displays installed snaps, versions, and channels.

References:

* [https://man7.org/linux/man-pages/man8/snap.8.html](https://man7.org/linux/man-pages/man8/snap.8.html)
