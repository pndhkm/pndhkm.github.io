# Listing Installed Packages

Package managers track installed versions and dependencies.

### List all packages

Input:

```
dpkg -l
```

Output:

```
ii  bash     5.1-6ubuntu1    amd64  GNU Bourne Again SHell
ii  coreutils 8.32-4.1ubuntu1 amd64  GNU core utilities
```

This lists installed packages, architecture, and summary.

References:

* [https://man7.org/linux/man-pages/man1/dpkg.1.html](https://man7.org/linux/man-pages/man1/dpkg.1.html)
