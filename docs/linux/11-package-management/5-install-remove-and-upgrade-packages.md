# Install, Remove, and Upgrade Packages

APT resolves dependencies and manages lifecycle operations.

### Remove a package

Input:

```
sudo apt remove nginx
```

### Upgrade installed packages

Input:

```
sudo apt upgrade
```

This fetches and applies updated package versions from repositories.
