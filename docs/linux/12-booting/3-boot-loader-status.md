# Boot Loader Status (systemd-boot)
Input:

```
bootctl status
```

Output:
```
System:
     Firmware: UEFI 2.9
     SecureBoot: disabled
Loader:
     Product: systemd-boot 254
```

Displays boot manager information on systems using systemd-boot.

References:
- https://man7.org/linux/man-pages/man1/bootctl.1.html
