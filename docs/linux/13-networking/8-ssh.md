# SSH

SSH provides secure remote shell access using encrypted TCP connections.

### Connect to remote host

Input:

```
ssh user@192.168.1.20
```

No output because SSH produces interactive session connection messages only.

References:

* [https://man7.org/linux/man-pages/man1/ssh.1.html](https://man7.org/linux/man-pages/man1/ssh.1.html)
