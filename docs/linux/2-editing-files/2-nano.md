# Nano

Nano is a simple, beginner-friendly text editor with visible shortcuts.

| Command | Description |
|----------|--------------|
| `nano file.txt` | Open or create a file |
| `Ctrl + O` | Save file |
| `Ctrl + X` | Exit |
| `Ctrl + W` | Search text |
| `Ctrl + \` | Replace text |
| `Ctrl + K` | Cut selected line |
| `Ctrl + U` | Paste text |

Example:

```
nano report.txt
```
After editing, press `Ctrl + O` to save, then `Ctrl + X` to exit.
