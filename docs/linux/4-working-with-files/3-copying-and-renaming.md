# Copying and Renaming

| Command           | Description             |
| ----------------- | ----------------------- |
| `cp fileA fileB`  | Copy a file             |
| `cp -r dir1 dir2` | Copy a directory        |
| `mv fileA fileB`  | Rename a file           |
| `mv file folder/` | Move a file to a folder |

Example:

```
cp notes.txt notes-backup.txt
```

Renaming:

```
mv oldname.txt newname.txt
```
