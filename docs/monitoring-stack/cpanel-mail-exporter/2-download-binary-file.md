# Download Binary File

You can download the prebuilt binary for your operating system from the [GitHub Releases page](https://github.com/pndhkm/cpanel_mail_exporter/releases/tag/v1.0.1).

### Available Binaries (v1.0.1)

- **macOS**
  - [Intel](https://github.com/pndhkm/cpanel_mail_exporter/releases/download/v1.0.1/cpanel_mail_exporter_1.0.1_darwin_amd64.zip)
  - [Apple Silicon](https://github.com/pndhkm/cpanel_mail_exporter/releases/download/v1.0.1/cpanel_mail_exporter_1.0.1_darwin_arm64.zip)
- **Linux**
  - [AMD64](https://github.com/pndhkm/cpanel_mail_exporter/releases/download/v1.0.1/cpanel_mail_exporter_1.0.1_linux_amd64.zip)
  - [ARM64](https://github.com/pndhkm/cpanel_mail_exporter/releases/download/v1.0.1/cpanel_mail_exporter_1.0.1_linux_arm64.zip)
- **Windows**
  - [AMD64](https://github.com/pndhkm/cpanel_mail_exporter/releases/download/v1.0.1/cpanel_mail_exporter_1.0.1_windows_amd64.zip)
  - [ARM64](https://github.com/pndhkm/cpanel_mail_exporter/releases/download/v1.0.1/cpanel_mail_exporter_1.0.1_windows_arm64.zip)
### Instructions

1. Download the zip file for your platform.
2. Unzip it:

```bash
unzip cpanel_mail_exporter_1.0.1_<platform>.zip
```
