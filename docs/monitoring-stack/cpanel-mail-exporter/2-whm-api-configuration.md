# WHM API Configuration

Advanced reference for enabling and validating WHM features that expose **cPanel API**, **Email Track**, and **Mail Troubleshooting** capabilities. 

### Feature Manager Activation (UI)

WHM → Manage API Token → Generate Token, then enable:

* Perform cPanel API and UAPI functions through the WHM API cpanel-api
* Track Email track-email
* Troubleshoot Mail Delivery mailcheck
